# Welcome to my first thought

This is an example blog post written in **Markdown**!

## Features

- Write in Markdown
- Automatic date sorting
- Easy to add new posts
- Builds to static HTML

You can add:
- **Bold text**
- *Italic text*
- `Code snippets`
- Links
- Images
- And more!

## How to add new posts

Just create a new `.md` file in `src/content/thoughts/` with the frontmatter at the top, and it will automatically appear on your site!
