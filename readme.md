#  Tech Stack

- Framework: https://astro.build v5.15.5

- Email: https://resend.com
- Analytics: Vercel Analytics & Speed Insights


Development

  npm install

### Start dev server
  npm run dev

### Build for production
  npm run build

### Preview production build
  npm run preview
